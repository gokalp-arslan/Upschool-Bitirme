# Kariyer ve CV Platformu

Bu proje `Next.js (App Router) + Tailwind CSS + Supabase` ile baslatildi.

## Kurulum

```bash
npm install
```

## Ortam Degiskenleri

`.env.local.example` dosyasini kopyalayip `.env.local` olustur:

```bash
copy .env.local.example .env.local
```

Sonrasinda Supabase degerlerini doldur.

## Calistirma

```bash
npm run dev
```
